# Основное задание
Нужно написать небольшую программу каталога товаров на Java.

Создай класс Money, который хранит цену (сумму и валюту).
Этот объект должен быть неизменяемым: после создания его нельзя менять.
Можно только создавать новый объект, например при сложении или вычитании цены.

Создай класс Product, который описывает товар:
 - id, name, price, category.

Он тоже должен быть неизменяемым.
Если нужно изменить цену или категорию — создаётся новый объект товара, а старый остаётся таким же.


Сделай интерфейс PriceValidator, который проверяет, правильная ли цена у товара.


Создай класс ProductCatalog, который хранит товары в Map.

Он должен уметь:
 - добавлять товар,
 - находить товар по id, 
 - искать товары по категории, 
 - искать товары по диапазону цен, 
 - возвращать список всех товаров (который нельзя изменить).


В классе Main покажи работу программы:
 - создай несколько товаров,
 - покажи, что при изменении цены создаётся новый товар, а старый не меняется,
 - попробуй изменить список всех товаров (это должно быть невозможно),
 - сделай поиск по категории и по цене.

Главная идея задания: показать, как делать неизменяемые объекты и как правильно разделять код на классы и интерфейсы.

## Дополнительное задание (Comparator)

Создайте Comparator<Product>, который будет задавать порядок сортировки товаров (например, по цене или по названию).

В классе ProductCatalog замените структуру хранения товаров:

Map<String, Product> на TreeMap<String, Product>

Передайте созданный Comparator в TreeMap через конструктор.

Остальная логика ProductCatalog должна продолжать работать с новой структурой хранения.

## Дополнительное задание (Iterator)

Реализуйте собственный итератор для каталога товаров, который будет перебирать товары в обратном порядке.

Создайте класс ProductReverseIterator, реализующий интерфейс Iterator <Product>

Итератор должен возвращать товары каталога начиная с последнего и заканчивая первым.

В классе ProductCatalog добавьте метод:

Iterator<Product> reverseIterator()

который возвращает экземпляр ProductReverseIterator.

В классе Main используйте этот итератор для обхода и вывода товаров каталога в обратном порядке.
